## ⚡ 三步构建 APK（无需安装任何软件）

### 第 1 步：上传代码到 GitHub（2分钟）

1. 访问 https://github.com/new
2. 创建新仓库：`FoodDecisionAssistant`
3. 解压 `FoodDecisionAssistant-Cloud.zip`
4. 将解压后的文件上传到 GitHub
   - 拖拽文件夹到 GitHub 网页，或使用 Git 命令

### 第 2 步：等待自动构建（3-5分钟）

上传后 GitHub 会自动开始构建：
- 点击仓库上方的 **Actions** 标签查看进度
- 绿色 ✓ 表示构建成功

### 第 3 步：下载 APK（1分钟）

1. 点击完成的构建记录
2. 滚动到页面底部的 **Artifacts**
3. 点击 **FoodAssistant-Debug-APK** 下载
4. 解压获得 `app-debug.apk`

---

## 📱 安装 APK

1. 将 APK 传输到安卓手机
2. 点击安装（允许"未知来源"）
3. 开始使用餐食助手！
